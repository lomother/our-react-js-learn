### 备注：

- 此学习计划时间设定仅作参考，以实际学习难度、内容量为准。

- 项目名中不含 React 的，并非 React 项目，可能是纯 JS 项目。若项目对其他框架或技术栈依赖重，则放弃。主要学 JS 和 React。

- 若项目对商业框架依赖重，放弃。若项目的依赖框架过老，放弃部署，仅阅读理解。

- 代码行数为JS代码行数，且仅作参考，因为有时项目仓库包含了开发人员的调试代码，对学习者意义不大。
